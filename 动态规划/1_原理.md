    递归
    F(n) = F(n-1) + F(n-2)
    F(0) = 0 
    F(1) = 1 
    
        缺点  
        时间复杂度 O(2^n)
        不平衡二叉树
        画图 
        重复计算
         
    记忆化
        所有状态记忆一次
        
        O(n)
        
    从下往上推
        省下记忆化     
        

    def fib(n):
        pre0,pre1=0,1
        for i in range(n):
            pre0,pre1 = pre1,pre0+pre1
        return pre1        
    print(fib(int(input())))
