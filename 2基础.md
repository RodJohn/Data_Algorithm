




# 时间复杂度

O(1)  只运行一次  时间复杂度为1   常数运算

O(n)
       运行n次
       for(int i =1 ; i < n ; i++ )
        
O(n^2)
       运行n的平方次
       for(n)
       嵌套for(n)
       
O(log(n))

       运行n次  log(2)n
       for(int i =1 ; i < n ; i *=2  )
